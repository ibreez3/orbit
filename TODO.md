# Orbit TODO（Roadmap）

> 以 `feature/macos-optimization` 分支为当前基线：已具备 SSH / SFTP / 监控 / 跳板机 / 服务器分组 / 凭据分组 / 本地加密 / 多 Tab。  
> 本 TODO 聚焦“下一步怎么更好用”：配置能力、数据库面板、自动化与批量效率等。

---

## P0（近期必做：闭环 + 省心）

- [ ] **配置中心（Settings）基础版**
  - [ ] 终端字体：字体选择、字号、字重（最小可用）
  - [ ] 终端主题：预置主题（至少 Catppuccin 系列）+ 主题切换
  - [ ] 光标样式/闪烁、行高、字距（可选）
  - [ ] 默认 shell / 环境变量（可选）
  - [ ] 快捷键设置（可选：先预置，后续开放自定义）
  - [ ] 监控刷新频率（例如 1s/2s/5s/10s）
  - [ ] SQLite 默认 LIMIT、SQL 超时、默认只读开关

- [ ] **Sidebar 服务器/分组快速搜索过滤**
  - [ ] ⌘+K 打开搜索框
  - [ ] 支持按名称/host/tag 搜索

- [ ] **配置导入/导出（JSON）**
  - [ ] 导出服务器/分组/凭据组
  - [ ] 导入合并策略（覆盖/新增/跳过）
  - [ ] 敏感字段策略：默认不导出明文；可选“二次加密导出”

- [ ] **SQLite 数据库面板（Remote Console，经 SSH 执行 sqlite3）**
  - [ ] DB Profile：关联服务器 + 远端 db 路径 +（默认只读）
  - [ ] 左侧：表/视图列表（含搜索）
  - [ ] 中间上：SQL 输入（⌘+Enter 执行、可取消）
  - [ ] 中间下：结果表格（行数/耗时/错误信息）
  - [ ] 右侧：表结构/DDL（`.schema` 或 `pragma table_info`）
  - [ ] 防护：默认 LIMIT、结果大小上限、超时控制
  - [ ] 可用性检测：远端无 sqlite3 / 无权限 / 文件不存在的友好提示

---

## P1（效率增强：把重复操作“一键化”）

- [ ] AI 助手（默认支持 OpenAI / Anthropic 协议）
  - [ ] Settings：Provider 选择（OpenAI / Anthropic）
  - [ ] API Key 本地加密存储（沿用 AES-256-GCM），支持一键连通性测试
  - [ ] Terminal：Ask AI（自然语言 → 命令建议），支持“复制/一键执行”
  - [ ] Terminal：Review（风险审计）：高危命令二次确认 + 回滚建议
  - [ ] SQLite 面板：NL → SQL、Explain SQL、Fix Error（基于错误信息建议修正）
  - [ ] 隐私策略：默认只发送用户选中的文本；禁止自动执行（必须显式确认）

- [ ] 命令片段 Snippets
  - [ ] 按分组管理
  - [ ] 一键插入 / 一键执行
  - [ ] 变量占位（如 {{host}} / {{user}}）

- [ ] 多服务器批量执行
  - [ ] 选择多台服务器执行同一命令
  - [ ] 并发上限、超时、失败重试
  - [ ] 结果聚合视图（按服务器分组展示）

- [ ] 本地端口转发管理（ssh -L）
  - [ ] 规则列表：新增/删除/启停
  - [ ] 端口占用检测
  - [ ] 跳板机链路兼容

- [ ] SFTP 体验增强
  - [ ] 传输队列（暂停/继续/重试）
  - [ ] 拖拽上传下载
  - [ ]（可选）并发传输与限速
