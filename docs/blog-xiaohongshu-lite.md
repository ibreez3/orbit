# 🚀 让 AI 帮我写了个 SSH 管理客户端

不满意市面上的 SSH 工具，于是让 AI 帮我写了一个。

用的 **OpenCode + GLM5.1**，从头到尾我自己没写过一行代码。

---

## 怎么做的？

**第一步：聊需求。** 一开始连要做什么都不清楚，就跟 AI 随便聊。几轮对话下来，需求清晰了：SSH 终端 + SFTP + 资源监控 + 密钥管理。

**第二步：让它造。** AI 自己规划路径，一轮轮迭代：骨架 → 终端 → 文件管理 → 监控 → 分组拖拽。我的工作就是跑一遍，有问题反馈，它改。

**第三步：打磨体验。** 比如分组选择器改了三版，最终变成输入即生效，一步到位。

---

## 做出来啥？

一个叫 **Orbit** 的桌面客户端：
- SSH 终端连接 + 实时流量统计
- SFTP 文件浏览上传下载
- CPU / 内存 / 磁盘实时监控 + 趋势图
- 服务器分组拖拽 + 密钥认证
- 跨平台，macOS / Linux / Windows

---

## 真实感受

效率大概是纯手写的 **3-5 倍**。需求梳理、查文档、跨语言开发的时间基本归零。

但不是万能的——产品决策得自己拍板，边界逻辑要人工验收，最终效果必须自己跑一遍看。

**一句话：AI 是引擎，你是方向盘。别等需求完美再动手，聊着聊着产品就出来了。**
